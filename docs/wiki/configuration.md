# Configuration Guide

This page explains where to change settings and what each settings area does.

## Where Settings Live

- Initial setup: when you first add Concierge.
- Concierge Global Settings: for home-wide sources and priorities.
- Concierge Integration Settings: for Concierge integration behavior.
- Room and merged-room setup: for day-to-day behavior tuning.

See:

- [Concierge Global Settings](global-settings.md)
- [Concierge Integration Settings](integration-settings.md)

## Concierge Global Settings

Concierge Global Settings affects your whole home:

- Weather source selection and priority.
- News source selection and priority.
- Alarm connection, when an alarm integration is installed.
- Home-wide defaults for summary behavior.

For details, see [Concierge Global Settings](global-settings.md).

## Concierge Integration Settings

Concierge Integration Settings controls integration-level behavior such as provider connections and archive behavior.

For media continuity and Follow-Me behavior, ensure the media provider and room playback targets are configured and valid.

For details, see [Concierge Integration Settings](integration-settings.md).

## Person Settings

Person settings help Concierge give the right answer to the right person:

- Person name and Home Assistant person
- Mobile target
- Communication preferences
- Calendar and email choices for that person
- Task and shopping list choices for that person

See [Initial Person Setup](person-setup.md).

## Room Settings

Room settings make local responses accurate:

- Room names and aliases
- Lights, speakers, TVs, dashboards, and other room devices
- Room style preferences

See [Initial Room Setup](room-setup.md).

## Merged Room Settings

Merged rooms let one request control multiple nearby spaces.

See [Merged Rooms](merged-rooms.md).

## Best Practices

- Keep names consistent.
- Keep room device lists up to date.
- Re-test key prompts after any major change.
- If results feel wrong, check person setup and room setup first.
