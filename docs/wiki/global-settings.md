# Concierge Global Settings

This page explains the Concierge Global Settings screen.

Concierge Global Settings is where you set home-wide behavior that affects every room and every person.

## What You Configure Here

Common Concierge Global Settings include:

- Weather source selection and weather source priority.
- News source selection and news source priority.
- Alarm integration connection (if an alarm integration is installed in Home Assistant).
- Home-wide behavior defaults used by Concierge when answering summary questions.

## Why This Page Matters

If Concierge Global Settings are not configured, Concierge may still work for basic room actions, but your broader household updates may be incomplete.

## Weather and News Priority

If you connect multiple weather or news sources:

1. Put your most trusted source first.
2. Keep the list short at first.
3. Test with a simple prompt such as: What should I know?

## Alarm Integration Connection

If you use an alarm system in Home Assistant:

1. Install and configure the alarm integration in Home Assistant first.
2. Open Concierge Global Settings.
3. Select the alarm entity you want Concierge to use.

If no alarm integration is installed, alarm controls in Concierge Global Settings may not appear.

## Relationship To Concierge Integration Settings

You can also view Concierge Integration Settings from the same management area.

- Concierge Global Settings: home-wide sources and priorities.
- Concierge Integration Settings: integration-level behavior and provider options.

See [Concierge Integration Settings](integration-settings.md).
