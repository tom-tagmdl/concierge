# Concierge Integration Settings

This page explains Concierge Integration Settings.

Concierge Integration Settings controls how the Concierge integration itself behaves.

## What You Configure Here

Typical integration settings include:

- Provider connections used by Concierge.
- Voice Identity linked setting (when Voice Identity is installed).
- Asset Intelligence provider setting (when Asset Intelligence is installed).
- Archive and retention settings.

For media continuity and Follow-Me behavior, provider connections and room playback targets must be configured and available.

## What This Is Not

Concierge Integration Settings is different from Concierge Global Settings.

- Use Concierge Global Settings for weather/news priorities and alarm connection.
- Use Concierge Integration Settings for Concierge integration behavior.

See [Concierge Global Settings](global-settings.md).

## Optional Integrations And Visibility

Some settings only appear if the related integration is installed:

- Asset Intelligence provider appears only when Asset Intelligence is installed.
- Voice Identity linked appears only when Voice Identity is installed.

If you do not see those options, install the related integration first.
