# Initial Person Setup

This page helps you set up each person in your household.

## Why This Matters

Good person setup helps Concierge:

- send messages to the right person
- use the correct calendar and email context
- give more useful personal responses

## Setup Steps

1. Create the person in Home Assistant if they are not already there.
2. Open Concierge person setup.
3. Select the matching person.
4. Set mobile notification target for that person.
5. Set calendar, email, task, and shopping choices for that person.
6. Save and test.

## Voice Identity (Optional)

If you use Voice Identity:

1. Turn on Voice Identity linkage in Concierge Integration Settings.
2. Attach the person to the correct voice profile.
3. Run voice enrollment if needed.
4. Test with a personal question like: What is on my calendar today?

To understand how recognition affects everyday Concierge behavior, see [Voice Identity Personalization](voice-identity-personalization.md).

## Quick Test Checklist

- The person receives messages on the expected mobile target.
- Personal calendar questions return the right events.
- Personal email summaries use the right mailbox.
- To-do and shopping responses match the right person.

## Follow-Me Identity Notes

Follow-Me media handoff depends on identity authority.

If identity is unresolved, low confidence, or conflicting, Follow-Me may refuse handoff by policy.

See [Follow-Me Media](follow-me-media.md).
