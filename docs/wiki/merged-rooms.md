# Merged Rooms

Merged rooms let Concierge treat nearby rooms as one shared space.

## When To Use Merged Rooms

Use this when you want one command to apply across more than one room.

Examples:

- kitchen + dining
- living room + den
- open floor living spaces

## Setup Steps

1. Create a merged room.
2. Add the rooms that belong in it.
3. Pick a primary room.
4. Save and test.

## Test Prompts

- What can I do here?
- Turn on the lights.
- Start movie time.

## Boundary: Merged Rooms vs Follow-Me

Merged rooms and Follow-Me are different behaviors.

- Merged rooms: one command can target grouped speakers across multiple rooms.
- Follow-Me: media can hand off from one room to another when transition and identity policy allow.

Follow-Me does not override merged-room grouping behavior.

See [Follow-Me Media](follow-me-media.md).

## If It Behaves Wrong

- Re-check which rooms are included.
- Re-check room device mappings.
- Re-test after any room rename or device move.
