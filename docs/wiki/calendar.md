# Calendar

This page helps you use calendar features in Concierge.

## What You Need First

1. Connect your calendar to Home Assistant using a supported calendar integration.
2. In Concierge person setup, make sure the correct calendar is selected for that person.

If Concierge does not know which calendar belongs to which person, personal calendar answers will be wrong.

## What You Can Ask

- What is on my calendar today?
- What is next on my schedule?
- Do I have anything important this afternoon?

## Best Practice

- Give each person their own calendar mapping in Concierge.
- Re-check mappings if you rename calendar entities or change accounts.

## If It Is Not Working

- Confirm the calendar integration is working in Home Assistant.
- Confirm the right calendar is selected in Concierge person setup.
- Ask a simple test question again.
