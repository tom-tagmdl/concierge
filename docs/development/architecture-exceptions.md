# Architecture Exceptions Register

Use this register only for temporary, explicit architecture exceptions.

## Active Exceptions

None.

## Exception Entry Template

- ID: `ARCH-EX-YYYYMMDD-<slug>`
- Rule: Which guardrail is being bypassed.
- Scope: Files or components affected.
- Justification: Why this cannot be implemented immediately.
- Owner: Accountable maintainer.
- Created: YYYY-MM-DD.
- Expiry: YYYY-MM-DD.
- Removal Plan: Concrete steps to remove the exception.
