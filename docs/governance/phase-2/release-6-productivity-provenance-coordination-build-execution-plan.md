## Ownership Boundary Review

Release 6 ownership boundaries:

- HTBW owns architecture, ADRs, contracts, models, governance, canonical definitions, provenance authority, household coordination authority where defined, and source-of-record/privacy boundaries.
- Concierge owns bounded consumption, resolution, planning, routing, orchestration, and execution behavior.
- External productivity systems own calendar/email/task/shopping/capture/knowledge records and lifecycles.
- Voice Identity owns attribution, confidence, enrollment, and lifecycle.
- Asset Intelligence owns asset/environment evaluation, significance, and metadata authority.
- Provenance owns no source records; it explains lineage and authority references.
- Household Coordination owns no source systems; it coordinates outcomes from authoritative sources.

Release 6 fails if any boundary above drifts.
